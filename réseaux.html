Réseaux informatiques

Un réseau informatique, c’est un ensemble d’ordinateurs et d’appareils qui peuvent communiquer entre eux et échanger des informations. Exemple : au collège, les ordinateurs, les imprimantes et l’ENT sont tous reliés.

À quoi ça sert ?

  • Partager des fichiers
  • Se connecter à Internet
  • Imprimer sur une imprimante commune
  • Gérer plusieurs ordinateurs depuis un serveur

Les éléments importants d'un réseau

  • Le routeur : distribue Internet
  • Le switch : connecte plusieurs appareils ensemble
  • Le serveur : stocke des données, comptes, fichiers
  • Les câbles ou le Wi-Fi : transportent l’information
  • Les ordinateurs / téléphones : les utilisateurs finaux

Schéma simple

      Internet
         |
      [Routeur]
         |
      [Switch] --- [Serveur]
         |
   -----------------------------
   |           |              |
[PC 1]     [PC 2]        [Wi-Fi]

Exemple dans mon stage

Chez TechMinds Alliance, j’ai appris comment un réseau fonctionne dans une véritable entreprise : connexion entre les postes, accès sécurisé, Wi-Fi privé pour les employés, et réseau séparé pour les invités.

Exercice (niveau 3ᵉ)

Relie chaque élément :

  • Serveur → Stocke les données
  • Routeur → Donne accès à Internet
  • Switch → Relie plusieurs appareils
  • Wi-Fi → Connexion sans fil
